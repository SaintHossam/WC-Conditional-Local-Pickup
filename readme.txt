Shows WooCommerce local pickup only when the shipping/billing city is Jeddah or Yanbu Al Bahr in Saudi Arabia.

== Description ==
Hides `local_pickup` for all Saudi cities except Jeddah (جدة/جده) and Yanbu Al Bahr (ينبع/ينبع البحر). Works on cart/checkout.

== Installation ==
1. Upload `wc-conditional-local-pickup-sa` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Test it by setting the shipping city to Jeddah or Yanbu Al Bahr.
